------------------------------------------------------------------------------------------
                   Readme for the CTLearn Package Oct. 20, 2014
------------------------------------------------------------------------------------------

The package includes the MATLAB code of algorithms CTLearn in [1,2]. 
It is a co-transfer learning framework that can perform learning simultaneously for multiple learning tasks.

[1]  Qingyao Wu, Michael Ng, and Yunming Ye, "Co-Transfer Learning Using Coupled Markov Chains with Restart", IEEE Intelligent Systems, DOI: 10.1109/MIS.2013.32, 2013
[2]  Michael Ng, Qingyao Wu and Yunming Ye. "Co-Transfer Learning via Joint Transition Probability Graph Based Method". In: SIGKDD Workshop on CDKD'12, pp.1-9, 2012 

We have included a sample data set "flowers-tree".
We have included an example to run the CTLearn algorithm. 
The parameters are set to the default values.
